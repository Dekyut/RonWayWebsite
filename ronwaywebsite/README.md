# RonWay Website

A lightweight React + Vite setup with a clean project structure for quickly iterating on the RonWay marketing site.

## Project structure

```
RonWayWebsite/
├─ public/
│  └─ index.html
├─ src/
│  ├─ assets/
│  ├─ components/
│  ├─ pages/
│  ├─ App.jsx
│  ├─ index.css
│  └─ main.jsx
├─ package.json
├─ vite.config.js
└─ README.md
```

## Getting started

```bash
npm install
npm run dev
```

The dev server opens automatically (or visit http://localhost:5173). Build for production with `npm run build` and preview the output via `npm run preview`.
